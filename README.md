## Installation

```bash
$ yarn install
```

## Running the app

https://docs.google.com/document/d/1sDpehJYU_0hc9WXgZzj6yGlgD5bCRc-D24nwcJiKrng/edit?usp=sharing

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
